## 📌 نسخه‌ی اول: مبتنی بر هوش مصنوعی (AI)

### 🚀 قابلیت‌ها
- تبدیل متن به وب‌سایت (مثل «نمونه‌کار سایبرپانکی»)
- تولید خودکار تصاویر مرتبط با Pollinations AI
- رابط هوشمند با نوار کناری، پیش‌نمایش موبایل و نمایش لایه‌ها
- خروجی یک‌کلیک به index.html

### 🛠️ تکنولوژی‌ها
- Python 3.13 + Google Gemini API (Flash) + Flask + TailwindCSS

### 💻 نحوه اجرا
ابتدا مخزن را کلون کنید:
git clone https://github.com/BehradB289/B2-AI-Web-Builder.git

سپس وابستگی‌ها را نصب کنید:
pip install google-generativeai flask flask-cors

کلید API خود را در فایل ai_builder.py قرار دهید.
(دریافت کلید از https://aistudio.google.com)

در نهایت موتور را اجرا کنید:
python ai_builder.py

## 🎨 نسخه‌ی دوم: دستی با دقت بالا (بدون AI) – حتی بهتر از نسخه AI

این نسخه کاملاً مستقل، بدون نیاز به Python یا API Key طراحی شده است.
با بهترین متریال کدنویسی (HTML، CSS، Tailwind، JavaScript خالص) ساخته شده و امکانات زیر را دارد:

- ⚡ بدون راه‌اندازی: فقط دوبار کلیک کنید – در هر مرورگری اجرا می‌شود.
- 🛠️ ویرایشگر بصری کشیدن و رها کردن (Drag & Drop) – جابجایی بخش‌ها، تغییر تم، ویرایش محتوا.
- 💾 ذخیره‌سازی خودکار در مرورگر (Local Storage)
- 🎨 تم‌های حرفه‌ای از پیش تعیین شده: مدرن، سایبرپانک، نوستالژیک، شرکتی.

فایل مورد نظر: B2-vs_standalone_builder.html

نحوه استفاده: فایل را دانلود کرده و با Chrome، Edge یا Firefox باز کنید. هیچ نصب یا سروری نیاز نیست.
