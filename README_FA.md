## 🎯 مشکل بزرگی که این پروژه حل می‌کند

ساخت یک وب‌سایت از صفر، ساعت‌ها زمان می‌برد. توسعه‌دهندگان مجبورند بارها کدهای تکراری برای نوار ناوبری، بخش اصلی، واکنش‌گرایی و چیدمان بنویسند.  
ابزارهای هوش مصنوعی موجود، سرعت را بالا می‌برند اما کنترل خلاقانه را از شما می‌گیرند. از طرف دیگر، طراحی دستی خالص زمان‌بر است و نیاز به تخصص بالایی دارد.

## 🚀 راه‌حل هیبریدی B2

این پروژه **دو مسیر مستقل و حرفه‌ای** برای ساخت وب‌سایت در اختیار شما قرار می‌دهد:

### ۱. 🤖 نسخه مبتنی بر هوش مصنوعی (برای سرعت)

- **تبدیل متن به وب‌سایت:** یک پرامپت ساده مانند *«یک نمونه‌کار سایبرپانکی»* را به کد HTML/CSS آماده تبدیل می‌کند.
- **تولید تصویر عصبی:** به طور خودکار تصاویر مرتبط با زمینه را از طریق Pollinations AI تولید می‌کند.
- **رابط هوشمند:** شامل نوار کناری واکنش‌گرا، نمایش لایه‌ها به صورت زنده و پیش‌نمایش موبایل.
- **خروجی تمیز:** با یک کلیک فایل index.html خروجی بگیرید و روی هر هاستی دیپلوی کنید.

**تکنولوژی‌ها:** Python 3.13 + Google Gemini 2.0 Flash API + Flask + TailwindCSS

---

### ۲. ✍️ نسخه دستی با دقت بالا (برای کیفیت و کنترل کامل)

این نسخه **کاملاً مستقل** است – بدون نیاز به Python، بدون نیاز به API key و بدون هیچ نصب و راه‌اندازی.  
با **بهترین متریال کدنویسی** (HTML خالص، CSS، Tailwind، JavaScript خالص) ساخته شده و امکانات زیر را دارد:

- ⚡ **بدون راه‌اندازی:** فقط فایل را دوبار کلیک کنید – در هر مرورگری اجرا می‌شود.
- 🛠️ **ویرایشگر بصری کشیدن و رها کردن (Drag & Drop):** جابجایی بخش‌ها، تغییر تم، ویرایش محتوا به صورت کاملاً بصری.
- 💾 **ذخیره‌سازی خودکار در مرورگر (Local Storage):** هرگز کارتان را از دست نمی‌دهید.
- 🎨 **تم‌های حرفه‌ای از پیش تعیین شده:** مدرن، سایبرپانک، نوستالژیک، شرکتی.
- 🏆 **چرا بهتر از نسخه AI است؟** چون با دقت بالا و کنترل کامل دستی ساخته شده و در بسیاری از موارد خروجی به‌مراتب ریزبینانه‌تر و قابل‌استفاده‌تری نسبت به خروجی هوش مصنوعی دارد.

**فایل:** `B2-vs_standalone_builder.html`

## 💻 نحوه اجرا (فقط نسخه هوش مصنوعی)

اگر می‌خواهید از نسخه هوش مصنوعی استفاده کنید:

1. مخزن را کلون کنید:
   git clone https://github.com/BehradB289/B2-Hybrid-Web-Builder.git

2. وارد پوشه پروژه شوید:
   cd B2-Hybrid-Web-Builder

3. وابستگی‌ها را نصب کنید:
   pip install google-generativeai flask flask-cors

4. کلید API خود را در فایل ai_builder.py قرار دهید.
   (برای دریافت کلید به https://aistudio.google.com مراجعه کنید)

5. موتور را اجرا کنید:
   python ai_builder.py

**برای نسخه دستی حرفه‌ای:** فقط فایل B2-vs_standalone_builder.html را دانلود کرده و در مرورگر باز کنید. تمام.
